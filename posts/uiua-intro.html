An intro to Uiua

Uiua is a stack-based, array-oriented programming language that uses glyphs to represent its built-in functions and operators. For example, a simple Uiua program might look like this: 

            Sum ← /+
            Avg ← ÷⧻⟜Sum
            Avg [1 2 3 4 5 6 7]
Let's break this down. The first line defines a function called Sum, that will run the /+ glyphs when called. The / glyph is a modifier called reduce, that takes a function as an argument, here the + function, and uses it to reduce an array. So the Sum function sums an array. The next line defines the Avg function, to take the average of an array. It starts by calling the Sum function on the values passed to it (uiua code runs right-to-left), with the ⟜ modifier applied to it.
